# AI助手功能使用说明

## 功能概述

AI助手模块集成了硅基流动API，提供以下功能：

1. **天气查询**：获取指定城市的实时天气信息
2. **穿衣建议**：根据天气情况推荐合适的穿着
3. **家居设置建议**：提供空调、加湿器等设备的智能设置建议
4. **综合建议**：提供生活小贴士和建议

## 硅基流动API配置

### 1. 获取API密钥

1. 访问 [硅基流动官网](https://cloud.siliconflow.cn/)
2. 注册账号
3. 登录后进入控制台
4. 在"API密钥"页面获取你的API密钥

### 2. 配置API密钥

有两种方式配置API密钥：

#### 方式一：环境变量（推荐）

在系统环境变量中添加：
- 变量名：`SILICONFLOW_API_KEY`
- 变量值：你的API密钥

**Windows系统设置方法：**
1. 右键"此电脑" → "属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
4. 在"用户变量"或"系统变量"中点击"新建"
5. 输入变量名和变量值
6. 点击"确定"保存

#### 方式二：代码配置

在 `aiassistant.cpp` 文件中修改以下代码：

```cpp
// 默认API密钥（请替换为你自己的）
m_apiKey = "your-api-key-here";
```

将 `your-api-key-here` 替换为你的实际API密钥。

## 使用方法

1. 启动应用程序并登录
2. 点击左侧菜单中的"🤖 AI助手"
3. 在输入框中输入城市名称（默认为北京）
4. 点击"获取建议"按钮
5. 等待AI生成建议（通常需要几秒钟）
6. 查看天气信息和智能建议

## API模型说明

当前使用的AI模型是 `Qwen/Qwen2.5-72B-Instruct`，这是一个强大的中文对话模型，能够理解复杂的指令并生成高质量的建议。

## 注意事项

1. **API配额**：硅基流动提供免费配额，但超过后需要付费。请关注你的API使用情况。

2. **网络连接**：AI助手需要稳定的网络连接才能正常工作。

3. **响应时间**：AI生成建议需要一定时间，通常为3-10秒，请耐心等待。

4. **城市名称**：请使用中文城市名称，如"北京"、"上海"、"广州"等。

5. **API密钥安全**：请妥善保管你的API密钥，不要将其提交到公开的代码仓库中。

## 故障排除

### 1. 网络连接失败

- 检查网络连接是否正常
- 确认防火墙没有阻止应用程序访问网络
- 尝试使用代理服务器

### 2. API密钥无效

- 确认API密钥是否正确
- 检查API密钥是否过期
- 确认API账户是否有足够的配额

### 3. 无法获取建议

- 检查城市名称是否正确
- 查看错误提示信息
- 确认网络连接状态
